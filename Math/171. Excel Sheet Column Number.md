Given a column title as appear in an Excel sheet, return its corresponding column number.

For example:
```
    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28 
    ...
```

**Example 1**:
```
Input: "A"
Output: 1
```

**Example 2**:
```
Input: "AB"
Output: 28
```

**Example 3**:
```
Input: "ZY"
Output: 701
```

**code:**

```js

/**
 * @param {string} s
 * @return {number}
 */
var titleToNumber = function(s) {
    if (s == null || s.length == 0) return 0;
    var ans = 0;
    for (let i = 0; i < s.length; i++) 
        ans = ans * 26 + (s.charCodeAt(i) - 64);
    
    return ans;
};


```
