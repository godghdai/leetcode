Given an integer, write a function to determine if it is a power of three.

**Example 1**:
```
Input: 27
Output: true
```

**Example 2**:
```
Input: 0
Output: false
```

**Example 3**:
```
Input: 9
Output: true
```

**Example 4**:
```
Input: 45
Output: false
```

**Follow up**:
Could you do it without using any loop / recursion?


**code 1:**
```js
/**
 * @param {number} n
 * @return {boolean}
 */
var isPowerOfThree = function(n) {
     return n > 0 && Math.pow(3,19) % n==0;
};
```

var isPowerOfThree = function(n) {
    if (n <= 0) return false;
